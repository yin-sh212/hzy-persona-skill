# CLAUDE.md

本文件为 Claude Code (claude.ai/code) 在此仓库中工作时提供指引。

## 项目方向

此仓库将构建为一个 Claude Code skill，将黄子洋的社交媒体内容蒸馏为可复用的数字人格，并支持生成特定风格的输出（网页 PPT、个人主页等）。

## 架构分层

### 第一层：自动化采集与推送（Python 脚本 + 定时调度）

Claude Code skill 本身是被动调用的，无法主动监控或推送。以下能力需要独立的 Python 脚本配合 cron / GitHub Actions：

- 定时爬取目标社交媒体平台的新内容
- 解析不同平台的数据格式，去重后增量写入语料库
- 检测到新内容后生成总结，推送到指定 QQ 群

### 第二层：Skill 本体（SKILL.md）

被调用时加载到 Claude 上下文中，负责：
- 根据语料库生成符合本人风格的输出
- 对语料进行分类、检索、摘要
- 响应用户的对话/生成请求（如"帮我写一段黄子洋风格的自我介绍"）

## 数据来源（目标平台）

- QQ 空间（大号 + 小号）
- 微信朋友圈
- 其他公开发布的社交媒体内容

## 语料分类体系

**主题分类**（一篇文章归属一个主分类）：

| 分类 | 范围 |
|------|------|
| 地球科学 | 学科讨论、竞赛经历、地科知识科普 |
| 自我成长 | 个人感悟、心理变化、阶段性反思 |
| 邓紫棋 | 演唱会记录、歌词引用、追星感悟 |
| 高考与升学 | 高考经历、中科大/南大/南信大申请、强基综评 |
| 旅游 | 长春、香港、南京等地的游记 |
| 大学日常 | 吉大生活、社交活动、节日记录 |
| 家庭 | 关于姐姐、父母、家乡（苏北）的内容 |
| 其他 | 不在以上分类的内容 |

**标签**（一篇文章可打多个，用于交叉检索）：`生病` `乡村振兴` `跨年` `中秋节` ...

## 原始语料说明

`raw/` 目录下为已有的原始中文语料：

- `1.txt` — 成长叙事，以 G.E.M. 歌词为线索串联高考、竞赛、大学生活
- `2.txt` — 思辨性文章，讨论地球科学与地理学的学科差异
- `3.txt` — 经验分享，地科竞赛参赛鼓励
- `4.txt` — 短篇随笔，关于长春天空的感悟

风格特点：第一人称、情感真挚、善用歌词/金句引用、叙事有节奏感。
