# 语料分类体系

## 分类原则

- **主分类**：一篇文章归属一个主分类，按内容的核心主题判定
- **标签**：一篇文章可打多个标签，用于交叉检索
- **时间索引**：每篇文章标注事件发生时间，支持按时间线浏览

## 主分类

| 分类 | 定位 | 典型内容 |
|------|------|---------|
| 地球科学 | 专业身份 | 地科竞赛、地联社团、地理知识、黄渤海分界线 |
| 行记 | 旅行记录 | 城市探索、风景、打卡 |
| 音乐 | G.E.M. + 孙燕姿 | 演唱会repo、歌词引用、专辑分享 |
| 随笔 | 个人感悟 | 回忆、自我剖析、内心戏，无法归入其他分类的 |
| 校园 | 吉大日常 | 社团活动、课程、同学社交 |
| 故土与家 | 家庭与家乡 | 苏北、家人、生日、节日 |
| 旧日 | 高考与竞赛往事 | 强基/综评/中科大/睢中/地科银牌 |

## 判定流程

1. 先按关键词匹配度打分，取最高分分类
2. 平局时按优先级：地球科学 > 旧日 > 故土与家 > 行记 > 音乐 > 校园 > 随笔
3. 无关键词命中时归入随笔（而非"其他"）
